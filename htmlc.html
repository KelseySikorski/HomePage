HTML Cheat Sheet




Basic tags

"html"- use this tag to create an HTML document.

"head"- use this tag after "html", and this where you can store links like CSS/ JAVASCRIPT.

"body" - What you reading is in the body.

HTML Elements

"h1"-"h6"- Header tags. H1 is the biggest fonts and H6 is the smallest.
Note: don't used "h6", it super small and it be hard to read.

"br"- line break.

"a"- anchor tag, connect link from other webpages.

"ul" and "ol"- unorder and order list.

"img"- links img to your webpage.
