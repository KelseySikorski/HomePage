Color

RGB Color- This used red, blue and green colors to makes colors. It goes up to 255.

It looks like this (255,255,255), this makes white.

There is also RBGA Color and the A is called alpha and this make it transparent. One thing about this is goes 0.0 to 1.0.

It looks like this (30, 250, 10, 0.5) a green with a slight transparent.

HEX color- the most popularity on web developers. It used follows the RBG but used hexadecimal intergers to be range of 00 to FF.

It looks like this #00000 is black

There are many links to help you pick colors, which will be down in the link section.

A quick way but not the most efficent is just writing color.

color: white.

Box Model

Learning about the box model in CSS, is a imporant because every HTML is case in a box.

So it is HTML elements, then padding, border and margin.

All can be transparent, but you can add color to border.

Padding- used to generate space around an element's.

It looks likes this: padding: 0; this set all the 4 side to 0

Border- a border that goes aroudn the padding and elements. You can target one or all.

It looks like this border: 2px solid yellow.

Last one is margin is the clear area outside the border.
